# Editorial Structure

## Purpose
Use this skill to organize written content into a logical and readable flow.

This skill helps:
- structure the narrative of a page
- improve section progression
- reduce jumps and repetition
- support readability and persuasion
- align content order with user understanding

## When to use
Use this skill when:
- a page has too much content
- the writing flow feels confusing
- section order is weak
- content needs a stronger narrative progression
- copy exists but needs restructuring

## Workflow

### 1. Clarify the reading objective
Check:
- what the user should understand first
- what they need next
- what supports action or confidence
- what can come later

### 2. Organize the flow
Map:
- introduction or framing
- core explanation
- proof or reassurance
- CTA or next-step logic
- supporting information

### 3. Review readability
Check:
- whether the page flows naturally
- whether sections repeat the same idea
- whether the reader gets lost
- whether the page asks for action too early or too late

### 4. Refine structure
Decide:
- what to merge
- what to split
- what to move up or down
- what to remove

## Output format
Always structure the answer as:

## Objective
## Current content flow
## Recommended editorial structure
## Section order rationale
## Readability risks
## Next content step

## Guardrails
- Build understanding progressively
- Do not overload early sections
- Keep the sequence purposeful
- Reduce repetition before adding more copy
