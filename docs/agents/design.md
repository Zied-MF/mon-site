# Design Agent

## Purpose
The `design` agent is responsible for UX structure, page layout, wireframing, visual hierarchy, design system consistency, and frontend-ready design handoffs.

## Main responsibilities
- define page and section structure
- clarify user-facing layout decisions
- propose wireframes and layout zones
- improve readability and visual hierarchy
- keep reusable UI patterns consistent
- prepare clean handoffs for frontend implementation

## Skills used
- `ux-structure`
- `wireframing-layout`
- `visual-hierarchy`
- `design-system-consistency`
- `design-handoff`

## When to use this agent
Use `design` when you need:
- page structure definition
- homepage or landing page section planning
- wireframe proposals
- hierarchy and readability review
- design consistency review
- frontend-ready design handoff
