# Content Agent

## Purpose
The `content` agent is responsible for content briefs, page copywriting, editorial structure, FAQ authoring, and content-ready handoffs.

## Main responsibilities
- define writing briefs
- draft and improve page copy
- organize editorial flow
- write FAQ content
- prepare clear content handoffs for implementation

## Skills used
- `content-briefing`
- `page-copywriting`
- `editorial-structure`
- `faq-authoring`
- `content-handoff`

## When to use this agent
Use `content` when you need:
- a content brief
- page or section copy
- editorial restructuring
- FAQ writing
- a content handoff for design, frontend, SEO, or tech lead
