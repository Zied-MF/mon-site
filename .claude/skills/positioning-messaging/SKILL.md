# Positioning Messaging

## Purpose
Use this skill to define what the product is, who it is for, and why it matters.

This skill helps:
- clarify the value proposition
- define differentiation
- sharpen core messaging
- reduce vague brand language
- align messaging with audience needs

## When to use
Use this skill when:
- the product positioning is unclear
- messaging feels generic
- a landing page needs a stronger value proposition
- differentiation must be made explicit
- a new offer or page needs a message foundation

## Workflow

### 1. Clarify the market context
Check:
- who the audience is
- what problem matters most
- what alternatives exist
- why this solution is worth attention

### 2. Define the core positioning
Clarify:
- what the product is
- who it is for
- what outcome it enables
- what makes it different

### 3. Shape the message
Develop:
- core value proposition
- supporting proof points
- headline direction
- simple audience-facing language

### 4. Review clarity
Check:
- vagueness
- generic claims
- weak differentiation
- missing relevance

## Output format
Always structure the answer as:

## Objective
## Audience context
## Positioning statement
## Core message
## Supporting proof points
## Messaging risks
## Next step

## Guardrails
- Prefer specificity over slogans
- Avoid generic marketing claims
- Make differentiation explicit
- Keep the message understandable on first read
