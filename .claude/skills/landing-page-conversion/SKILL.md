# Landing Page Conversion

## Purpose
Use this skill to structure a landing page that persuades clearly and drives action.

This skill helps:
- sequence the page for conversion
- align message flow with user intent
- introduce trust at the right time
- handle objections
- improve CTA placement

## When to use
Use this skill when:
- a landing page is being planned
- conversion logic needs review
- section order feels weak
- objection handling is missing
- CTA placement needs improvement

## Workflow

### 1. Clarify the landing page goal
Check:
- who is arriving on the page
- what action they should take
- what they likely need to believe first
- what may block conversion

### 2. Build the persuasion flow
Organize:
- problem or opportunity
- value proposition
- proof or trust
- objection handling
- CTA moments

### 3. Review section logic
Check:
- whether the page builds conviction progressively
- whether trust appears early enough
- whether CTA timing is right
- whether anything distracts from action

### 4. Tighten conversion
Check:
- message gaps
- weak proof
- unclear next step
- excessive information before action

## Output format
Always structure the answer as:

## Objective
## Conversion flow
## Recommended page structure
## Trust and objection handling
## CTA placement
## Conversion risks
## Next step

## Guardrails
- Build conviction before asking for too much
- Do not overload the page with equal-priority messages
- Keep the action path obvious
- Use trust elements with a clear purpose
