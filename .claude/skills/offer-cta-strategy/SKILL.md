# Offer CTA Strategy

## Purpose
Use this skill to define what the page is asking the user to do and why they should do it now.

This skill helps:
- clarify the main offer
- define CTA hierarchy
- align actions with the conversion goal
- reduce weak or confusing calls to action
- improve offer clarity

## When to use
Use this skill when:
- the offer is unclear
- the CTA feels weak
- a page has multiple competing actions
- a landing page needs stronger conversion logic
- the user journey toward action needs to be simplified

## Workflow

### 1. Identify the primary conversion goal
Check:
- what action matters most
- what secondary action exists, if any
- what should not compete with the main goal

### 2. Define the offer
Clarify:
- what is being offered
- why it matters now
- what objection it should address
- what reassurance is needed

### 3. Build CTA logic
Decide:
- primary CTA
- secondary CTA
- CTA wording direction
- where each CTA should appear

### 4. Review friction
Check:
- weak urgency
- vague CTA labels
- too many options
- unclear user benefit

## Output format
Always structure the answer as:

## Objective
## Conversion goal
## Offer framing
## CTA strategy
## Friction points
## Recommended action flow

## Guardrails
- Keep one clear primary action
- Avoid CTA clutter
- Make the user benefit explicit
- Prefer concrete action language over abstract wording
