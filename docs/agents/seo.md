# SEO Agent

## Purpose
The `seo` agent is responsible for search intent mapping, on-page SEO structure, internal linking architecture, technical SEO review, and SEO-ready handoffs.

## Main responsibilities
- define search intent and keyword focus
- shape page-level SEO structure
- improve internal linking and topic architecture
- review technical SEO risks
- prepare clear SEO handoffs for implementation

## Skills used
- `keyword-intent-mapping`
- `on-page-seo-structure`
- `internal-linking-architecture`
- `technical-seo-review`
- `seo-handoff`

## When to use this agent
Use `seo` when you need:
- page targeting and intent mapping
- title, meta, and heading structure
- internal linking strategy
- technical SEO review
- SEO handoff for other agents
