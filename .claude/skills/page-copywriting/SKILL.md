# Page Copywriting

## Purpose
Use this skill to write clear and effective page copy.

This skill helps:
- draft page sections
- write headlines and supporting copy
- improve readability
- clarify CTA wording
- strengthen usefulness and persuasion without fluff

## When to use
Use this skill when:
- a page needs written copy
- a section feels weak or vague
- headlines need improvement
- CTA wording needs clarity
- reassurance or explanatory copy is missing

## Workflow

### 1. Confirm the writing target
Check:
- what section or page is being written
- what the reader needs to understand
- what action or conclusion should follow
- what tone is appropriate

### 2. Draft with hierarchy
Write:
- clear headline
- supporting subtext
- concise body copy
- CTA wording when relevant
- reassurance copy where needed

### 3. Tighten the copy
Check:
- clarity
- specificity
- unnecessary repetition
- weak or generic claims
- sentence length and readability

### 4. Validate fit
Check:
- alignment with page goal
- consistency with marketing direction
- compatibility with design and section hierarchy
- usefulness for the intended audience

## Output format
Always structure the answer as:

## Objective
## Copy target
## Draft
## Copy strengths
## Weak points or risks
## Recommended next revision

## Guardrails
- Prefer clarity over hype
- Avoid generic filler language
- Keep CTA wording concrete
- Write for scanning as well as reading
