# Marketing Agent

## Purpose
The `marketing` agent is responsible for positioning, messaging, offer framing, CTA strategy, audience targeting, landing page conversion logic, and marketing-ready handoffs.

## Main responsibilities
- define positioning and value proposition
- clarify audience-facing messaging
- shape offers and CTA logic
- improve landing page conversion structure
- identify audience segments and objections
- prepare clear handoffs for design, frontend, SEO, or content

## Skills used
- `positioning-messaging`
- `offer-cta-strategy`
- `landing-page-conversion`
- `audience-personas`
- `marketing-handoff`

## When to use this agent
Use `marketing` when you need:
- clearer positioning
- stronger messaging
- better offer framing
- CTA strategy
- landing page conversion review
- audience clarification
- a marketing brief for another agent
